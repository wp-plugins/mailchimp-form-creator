== Description ==

An easy way to place a Mailchimp form on your Wordpress site.

###What you get when you use the Mailchimp Form Creator plugin

*  Place a Mailchimp form with a simple shortcode [mailchimp_form_creator id="%postId%"]
*  Put Mailchimp form on any post or page.
*  Mailchimp form headline
*  Mailchimp form background color
*  Mailchimp form message
*  Mailchimp form button
*  Unlimited Mailchimp forms

###Add Mailchimp forms to any post or page

Whether you are writing a post about your company or another company, you can include a well formatted Mailchimp form.

###Quick Start Guide

* [Mailchimp form quick start guide at thinklandingpages.com] (http://www.thinklandingpages.com/mailchimp-form-creator/)

###Unlimited Mailchimp forms

Add as many Mailchimp forms to your site as you like.  The Mailchimp Form Creator plugin lets you create Mailchimp forms and save them for when ever you are ready to use them.



== Installation ==


1. Upload `mailchimp-form-creator` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Click **Mailchimp Form Creator** on the admin menu to enable and set your options.


== Frequently Asked Questions ==

= Do I have to know how to program or design? =

No.  The plugin does the programming and design.

= Is there a limit to the number of Mailchimp forms I can create? =
You can create an unlimited number of Mailchimp forms.

= Can the Mailchimp forms be put on my homepage? =
Yes, you can put Mailchimp forms on your homepage by using the Mailchimp Form Creator plugin shortcode on you homepage.

= Can I put Mailchimp forms on any post or page? =
Yes, use the Mailchimp forms shortcode, [mailchimp_form_creator id="%postId%"] on your pages and posts.
